React Interview Prep: Junior Developer Summary

1. Core Concepts of React

  • React: JavaScript library for building UIs with components.
  • JSX: Syntax that allows HTML in JavaScript.
  • Functional Components: Components written as functions.
  • Virtual DOM: Lightweight copy of the DOM for faster updates.

2. Props vs State

  • Props: Read-only data from parent to child.
  • State: Mutable data managed by the component itself.
  • Passing Data: Use props to pass data to children.
  • Child to Parent: Use functions passed as props.

3. React Hooks

  • Hooks: Functions for using React features in functional components.
  • useState: Adds local state.
  • useEffect: Handles side effects like fetching.
  • useRef: Access DOM or persist values.

4. Rendering Concepts

  • Conditional Rendering: Use if or ternary.
  • Rendering Lists: Use .map() with a unique key.

5. Events and Forms

  • onClick: <button onClick={handleClick}>
  • Controlled Components: Input values from state.

6. Routing and Data Fetching

  • React Router: Client-side routing library.
  • Data Fetching: Use fetch inside useEffect.
