Essential React.js Interview Questions

1. What are the differences between a class component and a functional component?

Answer:

  • Class Component: Uses ES6 class syntax, has lifecycle methods, and can hold state.
  • Functional Component: Uses functions, is simpler, and uses hooks like useState and useEffect. 
// Functional
function Hello() {
  return <h1>Hello</h1>;
}

// Class
class Hello extends React.Component {
  render() {
    return <h1>Hello</h1>;
  }
}

2. What are React Hooks? Name a few commonly used hooks.

Answer:

Hooks are functions that let you use state and other React features in functional components.

  • useState – for state management
  • useEffect – for side effects like fetching data
  • useContext – to use React context
  • useRef – to access DOM elements or store values
  • useMemo / useCallback – for performance optimization

3. How does the virtual DOM work in React?

Answer:

  1. React updates the virtual DOM when state or props change.
  2. It compares the new virtual DOM with the old one (diffing).
  3. It finds and applies the minimal changes to the real DOM (reconciliation).

This makes React fast and efficient.
