What is the difference between em, rem, %, and px?

px: fixed
em: relative to parent
rem: relative to root
%: relative to container

What is specificity in CSS?

Determines which rule applies: Inline > ID > Class > Element

What are pseudo-classes in CSS?

Define special states like :hover, :first-child, etc.

Difference between position values?

relative: normal pos
absolute: to parent
fixed: to viewport
sticky: scroll hybrid

How does the CSS box model work?

Box = content + padding + border + margin

What are media queries?

Apply styles based on screen/device properties.

Difference: visibility hidden vs display none?

hidden: invisible, space kept
none: no space

Flexbox vs Grid?

Flexbox: 1D layout
Grid: 2D layout

How to center a div?

Use margin auto, flexbox, or transform
