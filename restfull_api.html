React + REST API Flashcards

What is a REST API?
REST API is a set of rules that allows client and server to communicate over HTTP using standard methods like GET, POST, PUT, DELETE.
Which HTTP methods are commonly used in REST APIs?
GET, POST, PUT, PATCH, DELETE
How do you fetch data from a REST API in React?
Use fetch() or libraries like axios inside useEffect to call the API when the component mounts.
What is the role of useEffect in REST API calls?
useEffect runs code (like API calls) after the component renders, ideal for fetching data when a component mounts.

How do you handle loading and error states in React when calling an API?

What is the difference between PUT and PATCH?
PUT replaces the entire resource, PATCH updates only the specified fields.
What are CORS issues and when do they occur?
CORS issues happen when a frontend app tries to access a backend on a different domain without proper headers.
